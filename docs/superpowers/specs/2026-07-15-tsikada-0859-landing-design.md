# ЦИКАДА 0859 — лендинг ночного квеста в Великом Новгороде

**Дата:** 2026-07-15
**Статус:** Approved

## Контекст и происхождение проекта

Изначальный запрос смешивал два несвязанных брифа: рекламный лендинг AUREN (AI-водитель для автомобиля, "HANDS OFF.", "ANYWHERE.") и лендинг квеста «ЦИКАДА 0859» в Великом Новгороде. Уточнено с пользователем: реальный продукт — **ЦИКАДА 0859**. Структура секций, техника скролла (sticky hero + видео-скраббинг через GSAP ScrollTrigger + Lenis) и общая кинематографичная подача взяты из брифа AUREN только как референс формата — весь бренд, текст и визуальная тема AUREN не используются.

Квест — секретная организация/протокол в стиле Cicada 3301: холодные анонимные послания, шифры, простые числа, координаты, ощущение отбора, а не развлечения. 859 — год первого упоминания Новгорода в летописи.

## Цель

Статический одностраничный сайт, который:
1. Создаёт атмосферу таинственной организации, а не рекламирует развлечение.
2. Не раскрывает механику, ответы, маршрут или реальные локации квеста — только концепцию.
3. Даёт два пути регистрации: форма на сайте (email) и (в будущем, отдельным шагом) Telegram-бот.

## Технический стек

- Статический HTML + CSS + vanilla JS.
- GSAP + ScrollTrigger — анимации по скроллу, pin+scrub видео и SVG-маршрута.
- Lenis — плавный (инерционный) скролл.
- Оба подключаются через CDN, без сборщика.
- Постеры видео — сгенерированные градиентные `.webp` (ImageMagick), с `preload="metadata"` на видео и `loading` где уместно для изображений.

### Структура файлов

```
index.html
styles.css
script.js
/assets
  /video      — night-road.mp4, city-drift.mp4 (заглушки; реальные файлы подкладывает пользователь)
  /posters    — сгенерированные градиентные .webp постеры (по одному на видео-секцию)
  /icons      — инлайн SVG (маркеры карты и т.п.)
```

### Обработка видео-заглушек

Реальных видеофайлов нет и генерировать их в этой среде нечем (нет ffmpeg/видео-модели). Решение:
- `<video>` ссылается на `assets/video/*.mp4` по «правильным» именам — пользователь позже подложит свои файлы без изменения разметки.
- У каждого видео задан `poster` — сгенерированный тёмный градиентный `.webp` с шумом/текстурой, чтобы секция выглядела законченной уже сейчас.
- JS проверяет `error`/`loadeddata` на видео: если файла нет, секция остаётся на постере (по факту так и будет, пока файлы не подложены); если файл есть — включается GSAP-скраббинг (`video.currentTime = progress * video.duration` внутри `ScrollTrigger` с `pin: true, scrub: true`).

## Секции

**Навигация** (фиксированная, полупрозрачная поверх видео): `ПРОТОКОЛ · КОНЦЕПЦИЯ · СВЯЗЬ · ВХОД`

1. **Hero (sticky, night-road.mp4, pin+scrub)**
   - Кикер (моно): `ЦИКАДА 0859 // ТРАНСМИССИЯ АКТИВНА`
   - Заголовок, fade-in по скроллу: **«МЫ ВИДЕЛИ ВАС.»**
   - Подпись: «Одна ночь. Один город. Один шанс быть услышанным.»

2. **Видео-секция «город» (city-drift.mp4, pin+scrub)**
   - Заголовок: **«ГОРОД СТАНЕТ ШИФРОМ.»**
   - Короткий текст про Великий Новгород как площадку квеста, без деталей маршрута.

3. **Сигнал / числовая станция**
   - Заголовок: `СЛУШАЙТЕ.`
   - Анимированная визуализация «аудио-волны» (SVG/canvas-бары на GSAP timeline) + бегущая строка случайных групп цифр в духе numbers station. Полностью визуальный эффект, без реального аудиофайла (переосмысление секции «голосовой ассистент» из исходного брифа AUREN).

4. **Карта маршрута**
   - Стилизованная SVG-карта (обобщённые контуры, не реальные адреса/названия).
   - 4–5 точек-чекпоинтов с нейтральными подписями («Точка I», «Точка II», ...).
   - Линия маршрута «прорисовывается» через анимацию `stroke-dashoffset` при скролле.
   - Никаких спойлеров: ни реальных адресов, ни подсказок/ответов.

5. **Концепция**
   - Переписанное, интригующее описание (замена сухого исходника):

     > 859 — год, когда имя Новгорода впервые прозвучало в летописи.
     > Мы не выбрали эту дату случайно.
     >
     > ЦИКАДА 0859 — не квест. Протокол.
     > Приглашение получают не все — только те, кто способен видеть шифр там, где другие видят просто улицу.
     >
     > Простые числа. Координаты без названий. Сообщения без подписи.
     > Этой ночью Новгород станет площадкой для тех, кто готов слушать город, а не просто гулять по нему.
     >
     > Мы не развлекаем. Мы отбираем.

   - Без деталей механики, ответов, точных точек.

6. **Регистрация**
   - Заголовок: `ПОДТВЕРДИТЕ ПРИСУТСТВИЕ.`
   - Форма: Email + кнопка `ОТПРАВИТЬ` (клиентская валидация email, submit-обработчик в `script.js` с явной точкой интеграции backend'а — реального эндпоинта нет).
   - Вторичная кнопка «Регистрация через Telegram» — ссылка-заглушка (`href="#telegram"` с `data-telegram-placeholder`), которая будет заменена на реальную ссылку на бота отдельным шагом после того, как бот будет создан. Не блокирует остальной сайт.

## Факты о квесте (дата, цена, длительность, возрастной ценз)

Данных от пользователя нет. Используются явные текстовые плейсхолдеры (например, `[ДАТА]`, `от [ЦЕНА] ₽`), которые пользователь заменит вручную перед публикацией. Это относится и к любым другим конкретным вводным (адрес точки сбора, номер телефона поддержки и т.п.).

## Визуальный язык

- Палитра: почти-чёрный фон (`#0a0a0c`), холодный сигнальный синевато-зелёный акцент, редкие вспышки тёплого янтарного как контраст.
- Типографика: моноширинный шрифт (тип JetBrains Mono) для кодов/чисел/кикеров, узкий контрастный гротеск/serif для крупных заголовков.
- Эффекты: лёгкий film grain overlay, анаморфные блики на видео-секциях, много негативного пространства, минимум декоративного цвета.

## Вне рамок текущей сборки

- Backend регистрации и обработка оплаты не реализуются.
- Telegram-бот не создаётся и не подключается — сайт содержит только заглушку-кнопку под будущую ссылку.
- Реальные видео/фото-ассеты не создаются — только структура файлов и заглушки, готовые к замене.

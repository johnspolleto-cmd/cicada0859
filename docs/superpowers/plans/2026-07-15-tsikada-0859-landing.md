# ЦИКАДА 0859 Landing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the static, scroll-driven one-page landing site for the ЦИКАДА 0859 night quest (Veliky Novgorod), per `docs/superpowers/specs/2026-07-15-tsikada-0859-landing-design.md`.

**Architecture:** Single static `index.html` with six pinned/scrubbed `<section>`s, one global `styles.css`, and five small classic (non-module) JS files each exposing a `window.Cikada*` namespace so their pure logic is also unit-testable from Node via `node:test`/CommonJS `module.exports`. `main.js` boots Lenis + GSAP ScrollTrigger and wires every feature module.

**Tech Stack:** Static HTML/CSS/vanilla JS. GSAP 3.12.5 + ScrollTrigger (same version) + Lenis 1.1.13, all via jsdelivr CDN. ImageMagick (`convert`) to generate placeholder poster `.webp` images. Node.js built-in `node:test` for pure-logic unit tests (no npm dependencies).

## Global Constraints

- All visible copy is in Russian. Tone: cold, anonymous, "you are being selected, not entertained" — never marketing/hype language.
- Nav labels are exactly: `ПРОТОКОЛ`, `КОНЦЕПЦИЯ`, `СВЯЗЬ`, `ВХОД`.
- Palette (exact hex): background `#0a0a0c`, foreground `#e8e6df`, dim foreground `#8b8a84`, signal accent `#4dffc4`, dim signal `#1f5c47`, amber accent `#ffb020`.
- Fonts: `JetBrains Mono` (kickers/mono/numbers), `Oswald` (display headings), `Inter` (body copy) — loaded via Google Fonts `<link>`.
- Never invent quest facts. Unknown values (date, price, duration, age limit, contact email/Telegram handle) are literal placeholders: `[ДАТА]`, `[ЦЕНА]`, `[EMAIL]`, `[TELEGRAM]`.
- Never reveal real addresses, puzzle answers, or exact quest mechanics anywhere in copy or map markup — the route map uses a generic, non-geographic SVG path and neutral labels (`Точка I`, `Точка II`, ...).
- No backend, payment, or Telegram bot integration in this plan — the waitlist form and Telegram button are front-end only, structured for a future hookup.
- Respect `prefers-reduced-motion: reduce` (disable/shorten animations).
- No bundler, no framework, no npm dependencies for the site itself — plain `<script>` tags, CDN links, Node's built-in test runner only for pure-logic checks.
- Every CDN `<script>` tag carries a real `integrity="sha384-..."` (computed from the actual served file, never fabricated) and `crossorigin="anonymous"`. If a version pin changes, its hash must be recomputed the same way before the tag is edited.

```js
function computeScrubTime(progress, duration) {
  const clamped = Math.min(Math.max(progress, 0), 1);
  return clamped * duration;
}

function initVideoScrub(sectionId) {
  const section = document.getElementById(sectionId);
  if (!section) return;
  const video = section.querySelector('video');
  if (!video) return;

  let scrubReady = false;

  const enableScrub = () => {
    if (scrubReady) return;
    scrubReady = true;
    video.pause();
    ScrollTrigger.create({
      trigger: section,
      pin: true,
      scrub: true,
      start: 'top top',
      end: '+=150%',
      onUpdate: (self) => {
        if (!video.duration) return;
        video.currentTime = computeScrubTime(self.progress, video.duration);
      },
    });
  };

  video.addEventListener('loadedmetadata', enableScrub, { once: true });
  video.addEventListener('error', () => {
    section.classList.add('section--poster-fallback');
  });

  if (video.readyState >= 1) {
    enableScrub();
  }
}

if (typeof module !== 'undefined' && module.exports) {
  module.exports = { computeScrubTime, initVideoScrub };
}
if (typeof window !== 'undefined') {
  window.CikadaVideoScrub = { computeScrubTime, initVideoScrub };
}
```

function generateSignalGroups(count, groupSize) {
  const groups = [];
  for (let i = 0; i < count; i += 1) {
    let group = '';
    for (let d = 0; d < groupSize; d += 1) {
      group += Math.floor(Math.random() * 10).toString();
    }
    groups.push(group.match(/.{1,2}/g).join(' '));
  }
  return groups;
}

function initSignalWave(sectionId) {
  const section = document.getElementById(sectionId);
  if (!section) return;

  const ticker = section.querySelector('.signal__ticker');
  if (ticker) {
    ticker.textContent = generateSignalGroups(24, 6).join('   //   ');
  }

  const canvas = section.querySelector('canvas.signal__bars');
  if (!canvas) return;

  const ctx = canvas.getContext('2d');
  const barCount = 48;
  const bars = new Array(barCount).fill(0).map(() => Math.random());
  let width = 0;
  let height = 0;

  const resize = () => {
    width = canvas.width = canvas.clientWidth;
    height = canvas.height = canvas.clientHeight;
  };
  resize();
  window.addEventListener('resize', resize);

  const draw = () => {
    ctx.clearRect(0, 0, width, height);
    const barWidth = width / barCount;
    bars.forEach((value, i) => {
      const barHeight = value * height;
      ctx.fillStyle = i % 7 === 0 ? '#ffb020' : '#4dffc4';
      ctx.fillRect(i * barWidth, height - barHeight, barWidth * 0.6, barHeight);
    });
  };

  gsap.ticker.add(() => {
    for (let i = 0; i < bars.length; i += 1) {
      bars[i] += (Math.random() - 0.5) * 0.15;
      bars[i] = Math.min(1, Math.max(0.05, bars[i]));
    }
    draw();
  });
}

if (typeof module !== 'undefined' && module.exports) {

}

  window.CikadaSignalWave = { generateSignalGroups, initSignalWave };
}

function computeCheckpointDelay(index, total, baseDelay) {
  if (total <= 1) return 0;
  return (index / (total - 1)) * baseDelay;
}

function initRouteMap(sectionId) {
  const section = document.getElementById(sectionId);
  if (!section) return;
  const path = section.querySelector('.route__path');
  const points = Array.from(section.querySelectorAll('.route__point'));
  if (!path) return;

  const length = path.getTotalLength();
  path.style.strokeDasharray = String(length);
  path.style.strokeDashoffset = String(length);

  gsap.to(path, {
    strokeDashoffset: 0,
    ease: 'none',
    scrollTrigger: {
      trigger: section,
      start: 'top 75%',
      end: 'bottom 40%',
      scrub: true,
    },
  });

  points.forEach((point, i) => {
    const delay = computeCheckpointDelay(i, points.length, 0.6);
    gsap.to(point, {
      autoAlpha: 1,
      duration: 0.5,
      delay,
      scrollTrigger: {
        trigger: section,
        start: 'top 60%',
      },
    });
  });
}

if (typeof module !== 'undefined' && module.exports) {
